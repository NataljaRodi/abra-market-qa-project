# 🛒 Abra Market QA Project

This project demonstrates my hands-on experience in manual and API testing of a marketplace web application during QA internship.

## 📌 About the Project

Abra Market is a B2B marketplace platform where:
- Suppliers upload products
- Sellers purchase products in bulk

## 🧠 What I Did

### 🔹 Frontend testing
- Created Mind Map
- Designed checklist for Registration
- Wrote test cases for Cart
- Performed manual testing

### 🔹 API testing
- Worked with Swagger
- Created API test cases
- Tested endpoints in Postman
- Validated responses
