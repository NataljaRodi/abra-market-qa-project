# 🛒 Abra Market QA Project

This project demonstrates my hands-on experience in manual and API testing of a B2B marketplace web application during QA internship.

---

## 📌 Project Overview

Abra Market is a B2B marketplace platform where:
- Suppliers upload and manage products  
- Sellers purchase products in bulk  

The platform includes key features such as:
- Registration & authentication  
- Product catalog and search  
- Shopping cart  
- Checkout and payments  

---

## 🧠 My Contribution

During this project, I worked on both frontend and backend testing:

- Designed test documentation (mind map, checklists, test cases)  
- Performed manual testing of key user flows  
- Identified and documented issues  
- Conducted API testing using Postman and Swagger  
- Validated responses, status codes, and data
